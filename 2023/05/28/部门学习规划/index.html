部门学习规划

一、 部门总体规划

​ 以项目式学习为主，竞赛为辅，充分调动部门成员的学习积极性。部门成员使用博客记录学习成果，方便交流与学习。

二、 实施方案

（一） 学习计划安排

1. 数据爬取

（1）搭建Python开发环境，安装Anaconda。

（2）学习 Python 基础知识，包括数据类型、控制语句、函数、文件操作、正则表达式、模块和库等。

（3） 学习 HTTP 协议基础知识，包括请求和响应的格式、状态码等。

（4） 学习使用requests库发送HTTP请求和处理响应。

（5） 学习正则表达式解析HTML页面。

（6） 编写爬虫程序，爬取网站疫情数据的列表并保存到本地文件。

（7） 学习使用Beautiful Soup库解析HTML页面。

（8） 学习使用Xpath解析HTML页面。

（9） 编写爬虫程序，爬取网站疫情的详细数据并保存到本地文件。

2. 数据整理、清洗

爬虫的数据清理和整理是非常重要的一步，可以提高数据的质量和可用性。因此，需要学习数据清洗、整理的有关知识，数据去重、数据筛选、数据格式化、数据清洗、数据归一化、数据合并等。

3. 数据挖掘、分析

爬虫的数据挖掘和分析是将爬取的数据进行深入分析和挖掘的过程，可以发现数据背后的规律和趋势，为后续的决策和应用提供支持。因此，需要学习数据可视化、数据分析、文本挖掘、时间序列分析、数据挖掘算法等。

4. 形成文档

将本次项目式学习的成果以文档形式记录下来，同时可锻炼部门成员的写作能力。

（二） 学习时间节点
